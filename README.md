# KIET Eid Greeting Card Generator

A simple, fast, and web-based application designed for the employees of the Karachi Institute of Economics & Technology (KIET) to generate personalized Eid greeting cards. 

## Overview

This tool allows staff and faculty to input their full name and designation, which are then dynamically overlaid onto a standard KIET Eid greeting template. The application runs entirely in the browser using HTML5 Canvas, meaning no backend server, database, or complex hosting is required.

## Features

* **Live Preview:** Users can see exactly how the greeting card will look before downloading.
* **Auto-Scaling Text:** Automatically adjusts the font size to ensure longer names or designations fit perfectly within the design without running off the edges.
* **Instant Download:** Generates a high-quality `.jpg` file ready to be shared on social media, WhatsApp, or via email.
* **KIET Branding:** The user interface features the official KIET color palette (Blue, Cyan, and Orange).
* **Zero Dependencies:** Built with pure HTML, CSS, and Vanilla JavaScript.

## How to Use (For Employees)

1. Visit the live URL.
2. Enter your **Full Name**.
3. Enter your **Designation**.
4. Click **Preview** to see your customized card.
5. Click **Download** to save the `.jpg` image to your device.

## Repository Structure

* `index.html`: Contains all the HTML structure, CSS styling, and JavaScript logic for the canvas rendering.
* `template.jpg`: The blank background image of the Eid poster. *(Note: This file must be present in the root directory for the app to function).*

## Deployment

This project is designed to be hosted seamlessly on GitHub Pages.
